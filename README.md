## 📌 Sobre o Projeto

Este projeto consiste no desenvolvimento de um **portfólio pessoal**, com o objetivo de apresentar minhas habilidades, projetos e informações profissionais de forma organizada e visualmente atrativa.

O portfólio funciona como uma vitrine online, permitindo que recrutadores e visitantes conheçam melhor meu trabalho e minha trajetória na área de tecnologia.

## 🎯 Objetivos

- Apresentar meus projetos desenvolvidos
- Destacar minhas habilidades técnicas
- Disponibilizar formas de contato
- Criar uma presença profissional na web

## 🛠️ Tecnologias Utilizadas

- ⚛️ HTML5  
- 🎨 CSS3  
- 💻 JavaScript
